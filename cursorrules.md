# Cursor Rules

- Em telas de CRM, todo filtro do tipo seleção deve usar o mesmo padrão visual do dropdown de Funil (botão com chevron + menu customizado), evitando `<select>` nativo para manter consistência de UX/UI.

- Em CRM Leads, filtros de seleção (Funil, Etapa, Origem, E-mail preenchido) devem seguir o padrão visual do dropdown de Funil (botão + chevron + menu customizado), evitando `<select>` nativo.
