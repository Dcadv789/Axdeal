# Regra de Espaçamento Horizontal (Global)

- Padrão oficial de espaçamento horizontal do conteúdo: **30px** em cada lado.
- Aplicação global via componentes compartilhados:
- `ecossistema_axory/components/ui/PageContainer.tsx`
- `ecossistema_axory/components/ui/navigation/SubSidebarLayout.tsx`

## Regra prática

- Todo conteúdo de página deve usar container compartilhado com `px-[30px]`.
- Em layouts com sub-sidebar, o conteúdo principal também deve usar `px-[30px]`.
